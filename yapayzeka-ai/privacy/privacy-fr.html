Politique de Confidentialité

Général
 
Pour utiliser le site web et/ou le service, nous collectons vos données personnelles. Les détails relatifs au traitement de vos données personnelles sont fournis dans cette Politique de Confidentialité.

Cette page vous informe de nos politiques concernant la collecte, l'utilisation et la divulgation de données personnelles lorsque vous utilisez ce site web (ci-après dénommé "vous" ou "utilisateur").

En utilisant le site web, vous acceptez la collecte et l'utilisation des informations conformément à cette politique. Sauf indication contraire, les termes utilisés dans cette Politique de Confidentialité ont les mêmes significations que dans nos conditions d'utilisation du site web.


Méthodes et principes de traitement des données personnelles

Nous attachons une grande importance à votre sécurité et traitons vos informations personnelles de manière respectueuse et conformément à cette Politique de Confidentialité et à la législation en vigueur. Si vous résidez dans l'Union Européenne, nous traitons vos données personnelles conformément au Règlement Général sur la Protection des Données (RGPD) de l'UE 2016/679.

Afin de prévenir toute divulgation ou utilisation non autorisée de données personnelles, nous avons mis en place des mesures techniques et organisationnelles adéquates pour protéger les données traitées contre toute violation de sécurité.

En cas de violation affectant vos droits et libertés, nous vous informerons immédiatement par e-mail.

Nous ne traiterons pas vos données personnelles sans votre consentement préalable.


Données personnelles traitées

Le traitement des données personnelles intervient après avoir complété le formulaire d'inscription sur le site web. En soumettant le formulaire, vous acceptez de conclure un accord avec nous pour faciliter la communication avec le fournisseur de produits et/ou services (Annonceur).

Nous collectons et traitons les adresses IP et les cookies des utilisateurs du site web. Les cookies sont de petits fragments de données envoyés par un serveur web et stockés sur votre ordinateur. Ces données nous aident à améliorer votre expérience sur le site web. Vous pouvez désactiver les cookies dans les paramètres de votre navigateur, mais restreindre leur utilisation peut affecter le fonctionnement du site web.

Vous garantissez que toutes les informations personnelles fournies sont actuelles, précises et exactes. Vous êtes responsable de tout dommage résultant de la fourniture d'informations personnelles incorrectes.


Méthodes de traitement des données personnelles

Le traitement des données personnelles est effectué automatiquement. Ce traitement ne peut entraîner aucune conséquence légale ni vous affecter de quelque autre manière.

Le traitement des données personnelles peut être délégué à d'autres fournisseurs de services d'email, processeurs et prestataires de services informatiques et cloud. La sélection des processeurs est effectuée avec soin pour garantir la protection de vos données personnelles pendant le traitement.

Vous devez fournir vos données personnelles aux Annonceurs pour utiliser le service.

Les données peuvent être transférées vers tout pays dans le monde. Les destinataires de vos données personnelles sont les Annonceurs et les processeurs. Le transfert est effectué avec des mesures adéquates pour garantir la protection de vos données personnelles.


Base juridique et durées de traitement des données personnelles

Pour vous fournir nos services, il est nécessaire de traiter les données personnelles fournies dans le formulaire d'inscription. Le traitement est effectué pendant la durée où les services vous sont fournis.

Les adresses IP, les e-mails et les informations dérivées des cookies sont traités pour vous envoyer des notifications liées aux services et produits du site web. Le traitement est basé sur votre consentement donné dans le formulaire d'inscription. Vous pouvez retirer votre consentement à tout moment.

Cette Politique de Confidentialité peut être modifiée ou complétée unilatéralement. La nouvelle version de la politique entre en vigueur dès sa publication sur le site web.

N'hésitez pas à nous contacter si vous avez besoin de plus d'informations.
